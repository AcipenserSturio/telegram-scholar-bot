# telegram-scholar-bot

This is a Telegram bot that acts as an alternate frontend to Google Scholar.

## Requirements

* Python >=3.11

* pdm (`pip install pdm`).

## Installation and usage

`pdm install`

`pdm run start`

For use on a server, add a systemd service for `pdm run start`.
